##  常用的jsDoc块级标签
```
/**
 @author:作者

 @argument:参数

 @augments：参数

 @class： 类

 @constant：常数

 @constructor：构造

 @constructs： 构造

 @default：默认值

 @deprecated： 声明不赞成使用该方法（类）

 @description：说明

 @example ：范例

 @external  (extends)： 扩展 ，继承

 @field：变量

 @fileOverview ：文件信息

 @function： 功能

 @inner 私有，内部

 @private : 私有，内部

 @ignore： 忽视

 @event：事件

 @version：版本

 @type：类型 描述预期的类型变量的值或返回值的函数

 @throws :可能抛出的异常

 @static： 静态，访问该变量不需要实例

 @since： 自从

 @see： 描述相关的资源

 @scope ：作用域

 @lends： 作用域

 @return ：返回值

 @returns ：返回值

 @requires： 描述必须需要的资源

 @public： 说明内在变量是公开的

 @property ： 属性

 @param：参数

 @namespace： 命名空间
 
 */
```
